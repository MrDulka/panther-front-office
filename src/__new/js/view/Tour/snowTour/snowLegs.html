  • Composites list
    • AOI Coverage - % of selected AOI covered by valid data (i.e. Snow, NoSnow, Cloud pixels)
    • Cloud-free coverage - % of selected AOI covered by cloud-free data (i.e. classified as Snow & NoSnow pixels)
    • Distribution of coverage - Class distribution of Snow, NoSnow, Cloud classes in % within the area covered by valid data

    In the bottom you can export the statistics for all available composites in a form of an Excel file or export composites in GeoTIFF data format.
